# Outputs

> Store the draft deliverables produced by skills for this matter in this folder
> (risk matrices, chronologies, draft letters, memos, checklists). Every file here
> is **draft legal work product for attorney review** — not legal advice and not a
> final document. Index each output in `skills_used.md` and map its statements in
> `citation_map.md`.

Suggested naming: `<NN>-<skill-slug>-<short-description>.md`
(for example `01-nda-review-risk-summary.md`).

No output in this folder is relied upon, sent, filed, or signed until the
supervising attorney has reviewed and adopted it in `attorney_review.md`.
