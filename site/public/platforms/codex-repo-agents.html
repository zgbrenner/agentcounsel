Codex / repo agents

Repo-based coding and legal agents — such as Codex — can use AgentCounsel directly from a checkout of the repository.

Setup

  1. Give the agent access to a checkout of the AgentCounsel repository.
  2. Point the agent at adapters/codex/AGENTS.md — the repo-agent instructions that explain how to work in the library.
  3. The agent routes a task with WORKFLOW_ROUTER.md to the narrowest relevant skill, or uses a command from COMMANDS.md.
  4. It reads the core/ rules and the selected SKILL.md, then follows the Workflow.
  5. It produces draft legal work product and completes the Attorney Verification Checklist.

Tips

  • COMMANDS.md maps slash-style commands (for example /contracts:nda) to skills.
  • The agent should select one narrow skill per task rather than loading the whole library.
