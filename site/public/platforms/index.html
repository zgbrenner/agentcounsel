Platform setup

How to use AgentCounsel in common AI environments. The library is plain Markdown, so it works anywhere an agent or person can read text.

  • ChatGPT Projects — ChatGPT Projects group related chats and let you attach files and custom instructions that apply to every conversation in the Project.
  • Claude Projects — Claude Projects keep a shared knowledge base and instructions across every conversation in the Project.
  • Gemini Notebooks — Use AgentCounsel with Gemini by adding the skill files as notebook sources, so the model can read and follow them.
  • Codex / repo agents — Repo-based coding and legal agents — such as Codex — can use AgentCounsel directly from a checkout of the repository.
  • One-off prompt use — For a single task in any AI assistant — no Project, no setup — paste one skill as a self-contained prompt.
