Gemini Notebooks

Use AgentCounsel with Gemini by adding the skill files as notebook sources, so the model can read and follow them.

Setup

  1. Create a notebook or workspace in your Gemini environment.
  2. Add AgentCounsel’s operating rules as sources: the six files from the repository’s core/ directory.
  3. Add the SKILL.md for the skill you need as another source — copy it with the Copy Full Skill button from its catalog page.
  4. Ask Gemini to follow the skill’s Workflow and Output Format, and provide the Required Inputs.
  5. Review the output against the skill’s Attorney Verification Checklist.

Tips

  • AgentCounsel also ships a Gemini CLI extension at adapters/gemini/. Installing the repository as an extension loads the operating model automatically.
  • Add only the skills you need as sources; the library is large.
