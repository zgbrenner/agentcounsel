Claude Projects

Claude Projects keep a shared knowledge base and instructions across every conversation in the Project.

Setup

  1. Create a new Project in Claude.
  2. Add AgentCounsel’s operating rules to the Project knowledge: upload the six files from the repository’s core/ directory.
  3. Open the skill you need from the skill index, copy its full SKILL.md with the Copy Full Skill button, and paste it into a chat — or add it to the Project knowledge.
  4. Provide the skill’s Required Inputs and ask Claude to follow the skill’s Workflow.
  5. Review the output against the skill’s Attorney Verification Checklist.

Tips

  • AgentCounsel ships a Claude Code plugin bundle at adapters/claude-code-plugin/ and a local-folder playbook at adapters/claude-cowork/ for Claude Code users.
  • Keep one skill active per task; the core/ rules can stay in Project knowledge.
