One-off prompt use

For a single task in any AI assistant — no Project, no setup — paste one skill as a self-contained prompt.

Setup

  1. Open the page for the skill you need from the skill index.
  2. Click Copy One-Off Prompt. The copied text wraps the skill with AgentCounsel’s operating rules.
  3. Paste it into any AI assistant.
  4. Provide the Required Inputs the skill asks for.
  5. Follow the Workflow, then review the output against the Attorney Verification Checklist.

Tips

  • For repeated work, prefer a Project or notebook so the operating rules persist across tasks.
  • Use the Copy Full Skill button instead if you want only the raw SKILL.md.
