ChatGPT Projects

ChatGPT Projects group related chats and let you attach files and custom instructions that apply to every conversation in the Project.

Setup

  1. Create a new Project in ChatGPT.
  2. Attach AgentCounsel’s operating rules: copy the six files from the repository’s core/ directory into the Project files, or paste them into the Project’s custom instructions. These safety rules apply to every skill.
  3. For a task, open the skill you need from the skill index, use the Copy Full Skill button on its page, and paste the skill into a new chat in the Project.
  4. Give ChatGPT the skill’s Required Inputs — the document, the facts, the client role, the jurisdiction.
  5. Follow the skill’s Workflow, then review the result against its Attorney Verification Checklist before relying on it.

Tips

  • Load one skill per task. Do not paste the whole library into a Project.
  • The core/ rules can stay attached to the Project and be reused across many tasks.
  • The Copy One-Off Prompt button gives you a single self-contained prompt that already includes the operating rules.
