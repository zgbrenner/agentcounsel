# Architecture Decision Records

This directory records why AgentCounsel's structure works the way it does —
not what to build next. Each record captures one existing decision: the
status, the context that drove it, the decision itself, and its
consequences. New records should follow the same compact shape; see
`0001-plugin-bundle-is-generated.md` for the template shape to copy.

Attribution: the record format is adapted from the MADR (Markdown Any
Decision Records) template, MIT/CC0-licensed. AgentCounsel's records are
written independently and are shorter than the full MADR template.
