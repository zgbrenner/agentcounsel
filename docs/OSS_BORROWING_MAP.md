# OSS Borrowing Map

This map records how AgentCounsel should learn from the listed open-source
repositories without copying unsafe or incompatible material. Licenses were
checked against GitHub repository metadata on 2026-05-28, with a second pass
for repositories that GitHub reported as `NOASSERTION` or no license file.

AgentCounsel is MIT-licensed. Direct copying or close adaptation should be
limited to license-compatible sources and must follow
`docs/THIRD_PARTY_ATTRIBUTION_POLICY.md`. This PR copies no external code,
prose, prompts, schemas, examples, or eval content, so `THIRD_PARTY_NOTICES.md`
does not need an update.

A later addition to this map (the rows below covering `eyecite`,
`reporters-db`/`courts-db`, `cuad`, the oneNDA/Bonterms/Common Paper open
standards, and `adr/madr`) does adapt category taxonomies and a template
shape under CC-BY-4.0, BSD-2-Clause, and MIT/CC0 terms, written independently
in AgentCounsel's own words. That addition's attribution entries are recorded
in `THIRD_PARTY_NOTICES.md`.
