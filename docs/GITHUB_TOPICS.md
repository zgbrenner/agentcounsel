# GitHub topics

Recommended GitHub repository topics for AgentCounsel. Topics make the project
discoverable and describe what it is. Set them under the repository's **About**
panel (the gear icon).


The first sixteen are the core set requested for launch; `attorney-supervised`
and `draft-work-product` are added because they describe the project accurately
and reinforce the safety posture.

## Keep topics accurate — do not overclaim

Topics are public claims about what the project is. Keep them honest:

- **Do** use topics that describe the project as it actually is: a Markdown-native,
  attorney-supervised library of legal *workflows* that produce *draft work
  product* for review, usable across multiple AI tools.
- **Do not** add topics that imply something AgentCounsel is not. Avoid, for
  example: `legal-advice`, `ai-lawyer`, `legal-services`, `automated-legal-review`,
  `verified-citations`, or anything suggesting it replaces a licensed attorney,
  constitutes legal advice, or verifies the law automatically.
- Platform topics (`chatgpt`, `claude-skills`, `cursor`, `codex`, `gemini`) reflect
  that AgentCounsel ships packs/adapters for those tools — keep them only while
  that remains true. AgentCounsel is independent and not affiliated with, endorsed
  by, or sponsored by any AI vendor.

When in doubt, prefer fewer, more accurate topics over more, looser ones.
